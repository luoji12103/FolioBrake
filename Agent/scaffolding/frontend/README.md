# Frontend scaffold

Create a Tauri + React + TypeScript project here:

```bash
npm create tauri-app@latest frontend
```

Recommended selections:

- Package manager: npm or pnpm.
- Frontend framework: React.
- Language: TypeScript.

The frontend should call the backend health endpoint at `http://localhost:8000/health` first, then implement pages described in `docs/11_frontend_gui_spec.md`.
